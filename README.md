# 🎓 Skillhub — Udemy Clone Platform

Skillhub is a fully responsive, front-end clone of the popular e-learning marketplace Udemy. Built using standard semantic HTML5 structures and custom layout CSS3 styling, this platform replicates the visual layout, card patterns, and core user-interface flows of a modern digital learning hub.

## 🚀 Live Demo

Explore the interface live on GitHub Pages: https://jagadeeswaran-k007.github.io/Udemy-clone-project/

## 🛠️ Built With

*   **HTML5** – Used for clean semantic structuring of complex grids, course layouts, and category panels.
*   **CSS3** – Applied for typography styling, user interface themes, flexible boxes (Flexbox), and hover state effects.

## 🌟 Key Features

*   **Responsive Homepage UI:** Replicates the recognizable landing page structure of an educational marketplace.
*   **Interactive Design:** Features fluid CSS hover states, polished text layouts, and visually distinct course card containers.
*   **Dynamic Layout Framework:** Built systematically to practice clean multi-column layouts and media scaling properties without external frameworks.

## 🎯 Purpose

This project was developed to strengthen my front-end web development skills by recreating a popular online learning platform. It helped me gain hands-on experience with page layouts, responsive design, styling techniques, and website structure using HTML and CSS.
